# atcp
Linux userspace TCP/IP stack.

L2 IO is Based on NetMap
##Usage:
* Install netmap, See [here](https://github.com/luigirizzo/netmap)
* Download atcp
* Compile and run
```
	cd atcp
    make
    ./build/atcp
```


##Support

* Neighbour subsystem
* ICMPv4 echo reply/request
