# atcp
Linux userspace TCP/IP stack.

Base on NetMap
##Usage:
* Install netmap, See [here](https://github.com/luigirizzo/netmap)
* Download atcp
* Compile and run
```
	cd atcp
    make
    ./build/atcp
```


##Support

* Basic arp send/request
* ICMPv4 echo reply
